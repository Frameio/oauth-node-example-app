# oauth-node-example-app

## Setup

```
$ npm install
```

## Usage

Edit the following values in `index.js` to match your OAuth app configuration.

 - `clientID`
 - `clientSecret`
 - `callbackURL`
 - `scopes`

Next, run the application.

```
$ node index.js
```

The application will be available at `http://localhost:5050`.
